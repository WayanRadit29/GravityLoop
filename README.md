**SwingBound** is a physics-based 2D platformer game inspired by *SWINGO*.  
The player controls a character who cannot walk or jump, but instead swings around using a grappling rope to navigate through platforms and reach the goal.
This project is developed using **Python** and **Pygame**, applying the principles of **Object-Oriented Programming (OOP)** for clean and modular game architecture.
